# Sakura+ 🌸

Sakura+ é um catálogo de animes fictício, desenvolvido com fins educacionais como exercício de CRUD em PHP puro.

## Membros

- Benjamin Felippe (Machikko)
- Gabriel Araujo (AraujoTheCreator)

## Objetivo 

Este projeto foi desenvolvido com o objetivo de praticar:

- Desenvolvimento Web
- HTML
- CSS
- PHP
- SQL
- MySQL
- phpMyAdmin
- XAMPP
- Estruturação de repositórios
- Upload e validação de arquivos

## Funcionalidades

- Landing page com vídeo de fundo.
- Catálogo de animes em cards, estilo miniatura de vídeo.
- Página individual de cada anime, com contagem real de visualizações (incrementada a cada acesso)
- Sistema de likes funcional
- Cadastro de anime com upload de imagem
- Edição de anime
- Exclusão de anime com confirmação
- Barra de pesquisa
- Totalmente responsivo


## Como usar

1. Crie o banco de dados executando o arquivo `database.sql` no MySQL/phpMyAdmin. A tabela `animes` é criada vazia — nenhum anime de exemplo é inserido.
2. Inicie o Apache e o MySQL pelo painel do XAMMP
3. Acesse `home.php` no navegador para a landing page.
4. Cadastre seus próprios animes pelo botão.
